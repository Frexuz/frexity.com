# HTML guidelines

## Quotes

HTML attributes should always use double quotes

```
# Bad
<div class='box'>A nice box</div>

# Good
<div class="box">A nice box</div>
```

## Self closing tags

Always use explicit single-tag closer

```
# Bad
<br>

# Good
<br />
```

## Inside spacing

No spaces after starting tag or before ending tag.

```
# Bad
<div class="box"> A nice box </div>

# Good
<div class="box">A nice box</div>

# Good multiline
<div class="box">
  A nice box
</div>
```
